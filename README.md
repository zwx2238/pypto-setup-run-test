# pypto-setup-run-test

Automated setup and benchmark for **PyPTO + AKG Agents** on Ascend NPU.

## Quick Start

```bash
# 1. Set required environment variables
export AKG_AGENTS_API_KEY="sk-..."
export AKG_AGENTS_BASE_URL="https://api.deepseek.com/beta"
export AKG_AGENTS_MODEL_NAME="deepseek-chat"

# 2. One-time setup: clone repos, install CANN/torch/pypto/akg_agents, generate .env
bash setup_and_run_test.sh

# 3. Run benchmark (sources .env automatically)
bash benchmark.sh
```

## What Each Script Does

### `setup_and_run_test.sh`

Full environment setup in an isolated, repo-local layout:

1. Create local runtime directories (venv, cache, isolated `$HOME`)
2. Clone PyPTO, AKG, KernelBench, KernelBench-pypto repos
3. Download PyPTO third-party dependencies
4. Install CANN toolkit locally and source its environment
5. Install torch + torch\_npu
6. Install PyPTO into the local virtualenv
7. Install AKG Agents and its dependencies
8. Validate LLM config (`AKG_AGENTS_API_KEY` / `AKG_AGENTS_BASE_URL` / `AKG_AGENTS_MODEL_NAME`)
9. Generate `.env` with all runtime environment variables
10. Run the default pytest target

### `benchmark.sh`

Loads `.env` generated by setup and runs `benchmark.py` with benchmark parameters.

### `benchmark.py`

Standalone KernelBench benchmark script supporting verification and profiling. Loads `.env` from the same directory automatically, so it can also be invoked directly:

```bash
.local/venv/bin/python benchmark.py --idx 19
```

## Required Environment Variables

These must be exported **before** running `setup_and_run_test.sh`:

| Variable | Description | Recommended |
|---|---|---|
| `AKG_AGENTS_API_KEY` | LLM API key | *(your key)* |
| `AKG_AGENTS_BASE_URL` | LLM API endpoint | `https://api.deepseek.com/beta` |
| `AKG_AGENTS_MODEL_NAME` | LLM model name | `deepseek-chat` |

## Directory Layout

After setup, the repo-local layout looks like:

```
pypto-setup-run-test/
├── setup_and_run_test.sh
├── benchmark.sh
├── benchmark.py
├── .env                        # generated, contains all runtime vars
├── repos/
│   ├── pypto/                  # PyPTO source
│   └── akg/                    # AKG source
│       └── akg_agents/
│           └── thirdparty/
│               ├── KernelBench/        # original torch reference
│               └── KernelBench-pypto/  # pypto kernels
└── .local/
    ├── venv/                   # Python virtualenv
    ├── toolkit/                # local CANN install
    ├── downloads/              # third-party packages
    └── state/home/             # isolated $HOME
```

## Common Options

```bash
# Use a pip mirror
bash setup_and_run_test.sh -i https://mirrors.aliyun.com/pypi/simple/

# Specify torch versions
bash setup_and_run_test.sh --torch-packages "torch==2.3.1 torch_npu==2.3.1"

# Editable PyPTO install with extra pytest args
bash setup_and_run_test.sh --pypto-install-mode editable -- --maxfail=1 -x

# Re-run test only, skip all install/clone steps
bash setup_and_run_test.sh --smoke-test

# Override device
bash setup_and_run_test.sh --device-id 1 --device-type a3

# Full help
bash setup_and_run_test.sh --help
```

## Re-running After Setup

Once `setup_and_run_test.sh` has completed successfully:

```bash
# Option A: use benchmark.sh
bash benchmark.sh

# Option B: smoke-test mode (re-runs the default pytest target)
bash setup_and_run_test.sh --smoke-test

# Option C: run benchmark.py directly
source .env
python benchmark.py --idx 19 --llm adaptive_search
```
